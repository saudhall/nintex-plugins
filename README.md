# Image Annotator Plugin for Nintex Forms (v5)

An advanced image annotation plugin built for **Nintex Forms Cloud** and also fully usable inside **SPFx (SharePoint Framework)** solutions.

This plugin allows users to upload, annotate, mark up, review and save images directly within Nintex Forms using a modern interactive editor experience.

Perfect for:

* Site inspections
* Audits
* Incident reporting
* Asset management
* Construction markups
* Design reviews
* Approval workflows
* Visual feedback collection
* Governance & compliance processes
